  1. Java 中是否可以重写一个 private 或者 static 方法？
  • 不能。方法覆盖属于运行时动态绑定的，static方法是编译时静态绑定的。private修饰的变量或方法只能在当前类中使用，其他类继承无法访问private变量或方法，无法覆盖。
