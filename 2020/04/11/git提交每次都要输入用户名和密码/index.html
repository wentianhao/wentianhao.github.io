    git提交每次都要输入用户名和密码
    2020/04/11 git git

    问题:使用git提交到github,每次都要输入用户名和密码，操作起来很麻烦。

    原因:在clone项目或上传项目的时候，使用的是https方式，而不是ssh方式，切换一下即可。

    选择ssh方式，复制url

    解决办法：

    1.查看是https方式还是ssh方式

    2.移除https的方式，换成ssh方式

    3.添加新的git方式：ssh方式

    4.查看push方式是否修改成功

    5.push测试是否需要用户名和密码
