    Let minsup = 50% 最小支持度

    5 * 50% = 2.5 -> 3

    出现次数大于等于3的这些项集就是频繁项集

    Beer:3 Nuts:3 Diaper:4 Eggs: 3

    关联规则是形如X->Y的蕴含式

    支持度是X 和 Y 同时出现的概率

    可信度 是 X发生的条件下Y发生的概率

    minconf = 50% 最小可信度是50%

    输入：数据库D，minsup,minconf
    输出：频繁项集

    通过扫描D得到1-频繁项集L1;

    }
